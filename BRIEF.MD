I am a hobbyist photographer. I shoot mostly landscapes and travel photos (mostly on hiking and backcountry trips). I want to build a custom photography portfolio for myself.

I want it to look pretty minimal. The photos should really be centre stage. I'm more inclined to go for a dark mode theme. 

I want a map as the main centerpiece - not a google maps style embedded map application but a stylish world map. I want there to be pins on the world map that open up galleries of places I've shot photos in. It is very important that this map has NO national borders. Only land/water boundaries should be visible.

Performance is very important to me. The website shouldn't take long to load and it should work well on mobile.

I want to tech stack to be as minimal and lightweight as possible.
